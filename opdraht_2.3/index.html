Waarom noem je de eerste HTML-pagina; index.html?

De eerste HTML-pagina van een website heet vaak index.html, omdat deze pagina automatisch wordt geladen wanneer je alleen de domeinnaam van de website in je browser invoert (bijv. www.example.com). Het is eigenlijk een soort "hoofdpagina" of startpunt van de website. De server herkent "index.html" als de standaardpagina die hij moet laten zien als er geen specifieke pagina is opgevraagd.

Met welke HTML versie werken we nu?

We gebruiken HTML5, de nieuwste versie.

Met welke CSS versie werken we nu?

We werken met CSS3.

Hoe kan het dan een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera)

Browsers verwerken HTML en CSS elk op hun eigen manier, waardoor kleine verschillen kunnen ontstaan.

Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

Google Chrome is het populairst (65-70%). Het gebruikt ongeveer 200-500 MB RAM per tabblad.
